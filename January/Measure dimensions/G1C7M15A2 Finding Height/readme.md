

## 🌐 Changing the Language

You can switch the language of the app between English (`en`) and Indonesian (`id`).

1.  Open the `data.js` file.
2.  Find this line of code near the top:
    ```javascript
    const currentLanguage = "id";
    ```
3.  To change the language to English, simply change `"id"` to `"en"`:
    ```javascript
    const currentLanguage = "en";
    ```
4.  Save the file, and the app's language will be updated.
